Working Process

On this page you can see how our team prepared the project step by step

For creating our device we used different components that were suitable for our project. In the following passage we are going to meticulously describe all the components' general description, purpose, and usage. What is more, in order to have some final look we designed PCB on which we soldered all the modules.

ESP32 (WiFi + Bluethooth/BLE)

​​For the TELLME device we used the ESP32 D Wroom model with WiFi and Bluetooth/BLE modules. It is a main component of the project and every other module handled by using it. We programmed our ESP32 with the help of Arduino IDE.

BLE (Bluethooth Low Energy)

Depending on the purpose of our application we used the BLE (Bluetooth Low Energy) protocol for communication via sender and device. BLE is designed for very low power operation. Transmitting data over 40 channels in the 2.4GHz unlicensed ISM frequency band. The main reason why we used BLE for our TELLME project is because it is compatible with both iOS and Android devices, so the user can easily leave a message no matter what type of phone he or she has.

WiFi

One of the functionalities of our device is to send an email to the owner of a house, because of this we needed to use the WiFi module. Apart from that, users at any time can specify a new WiFi name and password for the device. Since the name and password of the WiFi are stored in the separate file on the SD card, in the case of power outage WiFi setup information will not be lost and the user will not have to indicate everything from scratch. Our ESP32 already had an integrated WiFi module which works on 2.4GHz ISM frequency band, and all we had to do was to configure it.

Nextion touch screen

We are using the Nextion touch screen to display the messages that we get from a stranger. The main reason why we chose Nextion touch screen is that it provided all the necessary details that we needed our touchscreen to have. We are using UART Communication between the touchscreen and ESP32.

MicroSD Card Module

The microSD card module is compatible with the microcontroller systems like Arduino and is used for reading and writing the data on SD card. This module is useful for our project since we are required to work with the text messages and images that need to be stored to avoid data loss during power outage. We are using SPI communication between ESP32 and MicroSD card module.

RTC Module

We are using the RTC module to display time and date on the Nextion Display’s main screen. Apart from that, when the message is sent we are appending the text message with the exact time and date when it was received. On the Nextion display, besides the text message the user can also see the date and time of the incoming message.

SMTP (Simple Mail Transfer Protocol)

Also, for sending information via email we used the ESP-MAIL-Client library which is generally used in SMTP (Simple Mail Transfer Protocol). First, before using SMTP we created a separate mail address for ESP32 and after two step authentication we got the option to send mail via our application by using SMTP. So, in case of available WiFi, the user will receive a text message with a stranger's picture from the TELLME device.

Camera Module OV7670

The Camera module is used to take photos of the people who are trying to leave any information to the homeowner via BLE. We used the I2C protocol for communication with ESP32. Pictures resolution for TELLME devices is 160x120 which can be modified afterwards. Camera module will be placed inside the house somewhere near the peephole in order to take strangers' pictures appropriately.

PCB Design

The circuit was designed in Easyeda circuit and PCB design tool. As mentioned above, our project uses ESP32 WROOM microcontroller, OV7670 camera module, Nextion touch Screen, MicroSD module and Real Time Clock. The connections were made according to the codes written for each component in order to operate correctly.

Device Design

Our device will be attached to the main door and power supply will be necessary for the TELLME device to work. Users can easily check new received messages by LCD touchscreen after arriving at home, or at any time via email.
